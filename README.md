# Vim files
These are my vim files.

## Install
I use [ghar](https://github.com/philips/ghar) to "install" .vim and .vimrc to ~/ by:

    ghar add git@github.com:mlb-/vim_files &&\
    (cd ~/ghar/vim_files && git submodule update --init) &&\
    ghar install
